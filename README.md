# Solidus Verify

Decentralized KYC verification service. Verify once, use everywhere.

- **Domain:** verify.solidus.network
- **Status:** First product — active development
- **Docs:** `../../docs/product-strategy.md`, `../../docs/protocol.md`

## Overview

KYC-as-a-service for crypto exchanges, DeFi platforms, and fintech companies. Users complete verification once through the Solidus network and present cryptographic proof to any requesting service — without re-submitting personal documents.

## Key Features

- Email, phone, address verification
- Government ID verification (KYC Levels 1-3)
- Zero-knowledge age/location proofs
- Per-verification fee model + enterprise subscriptions
